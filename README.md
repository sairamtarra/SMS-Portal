# SMS Portal Web Application

A simple web application built using **JSP, Servlets, JDBC, and MySQL** that allows users to register, login, and send SMS messages.

## Features
- User Registration
- User Login
- Dashboard
- Send Messages
- Logout
- MySQL Database Integration

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat

## Project Structure
SMSPortal
│
├── controller
│   ├── LoginServlet
│   ├── RegisterServlet
│   ├── LogoutServlet
│   └── SendMessageServlet
│
├── dao
│   ├── DBConnection
│   ├── UserDAO
│   └── MessageDAO
│
├── model
│   ├── User
│   └── Message
