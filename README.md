# SMS Portal Web Application

A simple web application built using **JSP, Servlets, JDBC, and MySQL**.

## Features
- User Registration
- User Login
- Dashboard
- Send Messages
- Logout

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat

## Project Structure
src
 └── com.smsportal
      ├── controller
      ├── dao
      └── model
